# Green Guard

Green Guard is a full-stack web application developed to digitize the Estidama Pearl Building Rating System (PBRS) certification process.

## Features

- User Authentication
- Role-Based Access Control
- Application Management
- Document Upload
- Certificate Generation
- Progress Tracking

## Technologies

- React.js
- Node.js
- Express.js
- PostgreSQL
- REST API
